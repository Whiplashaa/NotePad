1、使用GitHub创建仓库
2、git clone `URL` 克隆仓库到本地
3、$ssh-keygen -t rsa -C "your email@.com"
    邮箱要改成注册Github时用的邮箱
4、打开秘钥注意事项：
    秘钥格式为`.pub`文件,文件名为`id_rsa.pub`
    在gitbash里输入命令`clip < ~/.ssh/id_rsa.pub`,然后回就复制到剪贴板了
5、在GitHub的Setting中把ssh秘钥复制过去
6、之后在Gitbash 中输入命令`ssh -T git@github.com`，之后再输入`yes`.会出现相应界面表示连接成功
7、设置账号和邮箱
    a、`git config --global user.name "名字"`名字最好和GitHub上一样
    b、`git config --global user.email "邮箱"`邮箱一定要是GitHub上的注册邮箱

完成上述之后  就可以用`git add .` `git commit -m "message"` `git push origin` 上传代码了
